# 气体安全报警监控系统上位机软件功能需求文档

## 1. 功能范围

- 软件支持一机一码授权，未授权不可使用。
- 支持 485 串口通讯接入设备（485 转 USB）。
- 支持网络 TCP 通讯接入设备（485 转 LAN）。
- 支持控制器、探测器、通讯端口、地图、用户等基础配置。
- 支持数据监控、地图监控、设备监控、曲线监控。
- 支持报警记录、运行记录、操作记录查询、导出、打印、删除和清空。
- 支持设备报警、故障、离线等状态提示。
- 支持报警弹窗、警情列表、红色闪烁、离线置灰等提醒方式。
- 支持外部风机、电磁阀、继电器、分区设备联动控制。
- 支持本地 API 接口，方便后续二次开发。
- 支持传感器寿命、校验周期、维护计划提醒。
- 支持数据备份与恢复。
- 支持大屏展示、全屏轮播、报警优先画面。
