# 账号交付说明

## 默认管理员

首次运行且数据库没有任何用户时，系统会自动创建默认管理员：

```text
账号：admin
密码：admin123
```

首次登录后必须修改默认密码，未完成修改不能进入主界面。

## 厂家密码找回

登录页提供“找回密码”。客户忘记账号密码时，可以由厂家或授权服务人员输入厂家密码，为指定账号重置一个新密码。

当前厂家密码：

```text
kunlian20131213
```

找回流程只重置密码，不会绕过登录进入系统。密码找回成功和失败都会写入操作日志，日志不会保存厂家密码或新密码内容。

## 授权关系

账号密码和软件授权是两条独立流程：

1. 软件必须先完成授权，登录页才允许进入主系统。
2. 授权通过后，使用管理员或操作员账号登录。
3. 首次默认管理员登录必须先改密码。
